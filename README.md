# 🗺️ Mapeamento da Cidade e Municípios de São Paulo

Projeto dedicado ao mapeamento geográfico da cidade de São Paulo e dos municípios do estado e bairros, reunindo informações de localização.

## Objetivo

O objetivo deste projeto é centralizar informações geográficas de forma organizada, facilitando o desenvolvimento de aplicações que necessitem de dados sobre:

* Cidade de São Paulo;
* Municípios do Estado de São Paulo;
* Bairros;

## Funcionalidades

* 📍 Mapeamento da cidade de São Paulo.

├── ├── Bairros
